The .newick file submitted should contain the name of a file containing the species (host) tree and the gene (parasite) tree in newick format, followed by the associations between their leaves (one line per association in the format gene:species).

        ((p18, p19) p4, ((p22, (p24, p25) p23) p20, (p26, ((p30, p31) p28, (p32, p33) p29) p27) p21) p5) p3;
